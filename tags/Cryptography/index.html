密码学原理

0x01

  • 五元组（P,C,K,E,D）

    • Plaintext
    • Ciphertext

  • 攻击模型

    • 唯密文攻击
    • 已知明文攻击
    • 选择明文攻击
    • 选择密文攻击（强度最大）

  • 密码分析：Kerckhoff假设

  • 古典密码（单射）

    • 代换（改变字符）
      • 实例
        • 棋盘密码
        • 凯撒密码
        • 曾公密码
      • 类别
        • 仿射密码
          • 单表代换
          • a=1，移位密码
            • 若密钥以相同概率随机使用，则具有完善保密性
          • a=1，b=3，凯撒密码
          • 不能抵抗唯密文攻击，根据密文得到置换
        • 希尔HILL密码
          • 多字母代换
          • 不能抵抗已知明文攻击，根据明文获得矩阵长度
        • 维吉尼亚密码
          • 多表代换
          • 不能抵抗唯密文攻击，根据密文得到密钥长度
            • kasiski法
            • 重合指数法（0.065）
    • 置换（改变位置）
      • 希尔密码是特殊形式的置换密码
    • 流密码
      • 维吉尼亚密码是短周期的同步流密码
      • 周期流密码
      • 非周期流密码
        • 弗纳姆Verman密码
          • 一次一密
      • 可用线性递推关系产生伪随机序列
        • LFSR线性反馈移位寄存器
        • 不能抵抗已知明文攻击，根据明文获得LFSR的结构
      • 同步流密码
        • 存在周期问题
      • 异步流密码
      • 自动密钥密码
    • tips
      • e频率最高
      • gcd最大公约数
      • -
      • 可用分组密码生成密钥
    • 本原多项式

0x02

  • 密码体系安全性的评判

    • 计算安全性

    • 可证明

      • 归约

    • 无条件安全

      • 提供无限计算资源也无法被攻破
      • 完善保密性
        • 不能通过密文得到明文
        • 先验概率= 后验概率
          • 先验概率，如全概率；后验概率，如贝叶斯公式
        • |K|>=|C|>= |P|，单射
        • 证明一次一密不可破解
          • 假设|K|=|C|=|P|，该密码体制是完善保密的，当且仅当密钥使用概率都是1/|K|,且PC存在唯一密钥，使得ek(x)=y

    • dk1(y)=x 且 dk2(y)=x 不能确定k1 = k2

      • 仿射
      • playfair密码
